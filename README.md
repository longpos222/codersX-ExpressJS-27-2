Resource|POST<br>create|GET<br>read|PATCH<br>update|DELETE<br>delete|
:-|:-:|:-:|:-:|:-:|
`/auth/login`|log in|-|-|-|

`/transactions`|add new |get all|-|-|
`/transactions/{tranx-id}`|-|get info|update status|delete|
